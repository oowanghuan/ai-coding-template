## ⚠️ 重要约束

### Module ID 引用规则

> **强制**：本文档中所有模块引用必须使用 `03_MODULE_DECOMPOSITION.md` 中定义的 `module_id`。

**正确格式** ✅：
```
M001 user-auth
M002 user-profile
```

**错误格式** ❌：
```
用户认证模块        ← 禁止自由文本
登录功能            ← 禁止自由文本
auth                ← 禁止缩写
```

**为什么？**
- 确保 Roadmap 与 Module Decomposition 保持一致
- 支持自动化工具进行交叉验证
- 避免歧义和脱节
