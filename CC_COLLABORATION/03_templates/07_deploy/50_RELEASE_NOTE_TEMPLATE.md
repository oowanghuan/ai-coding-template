### 8.1 部署前检查

- [ ] 数据库迁移脚本已准备

- [ ] 回滚方案已准备

### 8.2 部署步骤

```bash
# 1. 拉取最新代码
git pull origin main
git checkout v{x.y.z}

# 2. 安装依赖
npm install

# 3. 执行数据库迁移
npm run migrate

# 4. 构建项目
npm run build

# 5. 重启服务
pm2 restart app
```

### 8.3 回滚方案

```bash
# 回滚到上一版本
git checkout v{x.y.z-1}

npm run build
pm2 restart app

# 回滚数据库（如需要）
npm run migrate:rollback
```
